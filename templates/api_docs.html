⚡ Overview

The YouTube Transcript API allows you to programmatically extract timestamped subtitles, plain text, and SRT files from any YouTube video. It supports fetching specific languages and automatic translation.

📡 Fetch Transcript

GET /api/v1/transcript

Fetches the transcript for a given YouTube video URL.

Query Parameters

Parameter Description
url required Full YouTube video URL (e.g. https://youtube.com/watch?v=...).
lang optional Language code to fetch (e.g. en, hi, es). Defaults to English if available.
translate optional Language code to translate the transcript into automatically.

Headers

X-API-Key Your API key (generated after payment). If omitted, falls under free tier (5 credits/day).

Example Request (Python)

import requests

url = "http://localhost:5000/api/v1/transcript"
params = {
    "url": "https://www.youtube.com/watch?v=aDhyAkXiDOY",
    "lang": "en",
    "translate": "hi"
}
headers = {
    "X-API-Key": "your_api_key_here"
}

response = requests.get(url, params=params, headers=headers)
print(response.json())

Example Request (JavaScript)

const url = 'http://localhost:5000/api/v1/transcript?url=YOUTUBE_URL&translate=es';

fetch(url, {
    headers: { 'X-API-Key': 'your_api_key_here' }
})
.then(res => res.json())
.then(data => console.log(data));

⏱ Rate Limits

To prevent abuse, the API enforces rate limits based on your plan:

  • Free: 5 credits/day
  • Basic: 50 credits/day
  • Pro: 500 credits/day + API key
  • Unlimited: Unlimited + API key

If you exceed your limit, the API will return a 429 Too Many Requests status code.
